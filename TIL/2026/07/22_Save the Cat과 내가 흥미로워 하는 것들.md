# 2. 오늘 배운 것

블레이크 스나이더의 『세이브 더 캣!(Save the Cat!)』은 복잡하고 모호한 작법 이론을 걷어내고, 철저하게 '흥행하는 상업 영화'를 만들기 위해 쓰인 할리우드의 실전 매뉴얼입니다.

당장 기획 중인 소설, 웹툰, 시나리오에 적용할 수 있도록 가장 핵심적인 3가지 도구를 정리해 드릴게요.

---

## 1. '세이브 더 캣'의 법칙 : 주인공에게 정을 붙이게 하라

책의 제목이기도 한 이 법칙은 "관객이 주인공을 응원하게 만들려면, 등장 직후에 '나무에 올라간 고양이를 구하는 것' 같은 작은 선행을 보여주라"는 뜻입니다.

* **실전 적용:** 주인공이 제아무리 성격이 괴팍하고 도덕적 결함이 있더라도(예: 사기꾼, 킬러), 초반 10페이지 안에는 그를 미워할 수 없게 만드는 인간적인 장면을 반드시 하나 넣으세요.

## 2. 로그라인(Logline) 필수 조건 : 한 줄로 유혹하라

잘 팔리는 이야기는 단 한 문장(로그라인)만 들어도 머릿속에 영화 한 편이 그려집니다. 실전에서 로그라인을 짤 때 다음 4가지가 들어있는지 점검하세요.

1. **아이러니 (Irony):** 설정 자체가 흥미를 유발하는 모순이 있는가? (예: *결벽증에 걸린 강박 통제광 형사가 쓰레기장에 잠입한다*)
2. **매력적인 그림 (A Compelling Mental Picture):** 문장을 읽자마자 어떤 사건들이 벌어질지 상상이 가는가?
3. **타깃 관객과 제작비 (Audience and Cost):** 누가 볼 이야기이며, 돈이 얼마나 들지 견적이 나오는가?
4. **죽여주는 제목 (A Killer Title):** 제목만으로도 이야기의 핵심이 전달되는가?

---

## 3. 실전 뼈대 잡기 : 15 비트 시트 (The 15 Beat Sheet)

이 책의 하이라이트입니다. 총 110페이지 분량의 시나리오를 기준으로, 어느 페이지(타이밍)에 어떤 사건이 터져야 하는지 정확히 짚어주는 템플릿입니다. **글을 쓰기 전 이 15칸부터 채워보세요.**

### 1막 : 설정과 출발 (1~25%)

| 비트명 (진행도) | 실전 작성 포인트 |
| --- | --- |
| **1. 오프닝 이미지 (1%)** | 주인공의 현재 상태를 보여주는 강렬한 첫 장면. 결말과 대조되어야 합니다. |
| **2. 주제 명시 (5%)** | 조연이 주인공에게 툭 던지는 대사 속에 이 이야기의 진짜 '주제'가 숨어 있습니다. |
| **3. 설정 (1~10%)** | 주인공의 일상과 부족한 점(결핍)을 보여줍니다. 주인공을 구속하는 답답한 환경입니다. |
| **4. 촉매제 (10%)** | 평화로운 일상을 박살 내는 사건이 터집니다. (예: 초대장, 살인 사건, 시한부 선고) |
| **5. 토론 (10~20%)** | 주인공이 변화를 거부하며 망설이고 갈등합니다. "진짜 할 거야?" 묻는 단계입니다. |

### 2막 : 갈등과 위기 (25~75%)

| 비트명 (진행도) | 실전 작성 포인트 |
| --- | --- |
| **6. 2막 진입 (20%)** | 주인공이 마침내 결심을 하고 새로운 세계(모험)로 발을 내딛습니다. |
| **7. B스토리 (22%)** | 메인 사건과 별개로 러브스토리나 멘토와의 관계 등 '주제'를 다루는 서브 플롯이 시작됩니다. |
| **8. 재미와 농담 (20~50%)** | 포스터나 예고편에 쓰일 만한 장면들입니다. 설정된 세계관의 재미를 본격적으로 보여줍니다. |
| **9. 중간점 (Midpoint) (50%)** | 주인공이 가짜 승리를 거두거나(다 잘될 것 같음), 가짜 패배를 겪습니다. 여기서부터 긴장감이 확 올라갑니다. |
| **10. 악당의 접근 (50~75%)** | 상황이 악화됩니다. 외부의 적이 압박해 오고, 주인공 내면의 문제도 터지기 시작합니다. |
| **11. 모든 것을 잃다 (75%)** | 최악의 상황. 주인공이 가장 의지했던 멘토가 죽거나, 치명적인 오해로 모든 관계가 파탄 납니다. 죽음의 기운이 맴돕니다. |
| **12. 영혼의 어두운 밤 (75~80%)** | 주인공이 바닥을 치고 절망하는 단계입니다. "난 끝났어"라고 자책하지만, 곧 깨달음을 얻습니다. |

### 3막 : 해결과 변화 (80~100%)

| 비트명 (진행도) | 실전 작성 포인트 |
| --- | --- |
| **13. 3막 진입 (80%)** | B스토리(조력자/사랑)에서 얻은 깨달음을 통해 주인공이 해결책을 찾아냅니다! |
| **14. 피날레 (80~99%)** | 주인공이 배운 것을 총동원해 악당(문제)을 물리칩니다. 주인공의 결핍이 완벽히 치유되고 낡은 세상은 무너집니다. |
| **15. 파이널 이미지 (99~100%)** | 오프닝 이미지와 완벽하게 대조되는 장면. 주인공이 얼마나 성장했는지 시각적으로 증명하며 끝납니다. |
