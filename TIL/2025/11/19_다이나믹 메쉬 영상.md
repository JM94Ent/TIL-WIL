모델링 모드에서 **다이나믹 메쉬(Dynamic Mesh) 변형**을 제대로 다루는 영상들만 골라서 몇 개 정리해볼게.
(주로 UE5 기준, 영어 튜토리얼이지만 화면 따라가면 충분히 이해 가능)

---

## 1. 다이나믹 메쉬 변형을 집중적으로 다루는 튜토리얼

### 1) **Unreal Engine 5 Tutorial – Modelling Mode Part 3: Deforming Meshes** (Ryan Laley Games) ([유튜브][1])

* **핵심 키워드**: Modeling Mode, **Deforming Meshes**, 편집/변형
* **내용**

  * UE5 모델링 모드 소개 시리즈 중, **메쉬 변형(Deform)** 파트만 따로 다룬 에피소드.
  * Dynamic Mesh를 대상으로:

    * 버텍스/엣지/폴리곤 선택
    * 이동/회전/스케일
    * 기본적인 변형 툴 사용법
    * 간단한 폴리곤 편집으로 형태 바꾸기
  * “정적인 스태틱메쉬 가져와서 수정”보다는 **에디터 안에서 바로 형태를 바꿔나가는 플로우**에 초점.
* **추천 이유**

  * 영상 제목부터 “Deforming Meshes”라 다이나믹 메쉬 변형 파트만 보고 싶을 때 딱 맞음.
  * 모델링 모드 UI, 툴 선택, 적용/Accept 버튼 쓰는 법 등 기본 개념을 한 번에 볼 수 있음.
* **찾는 법**

  * 유튜브에서: `Unreal Engine 5 Tutorial Modelling Mode Part 3 Deforming Meshes` 검색.

---

## 2. Dynamic Mesh Actor를 실제로 써보는 영상

### 2) **UE5 Modeling Mode – Dynamic Mesh Actors** (Ryan Schmidt) ([유튜브][2])

* **핵심 키워드**: **Dynamic Mesh Actor**, Modeling Mode
* **내용**

  * UE5에서 **Dynamic Mesh Actor를 활성화하고, 모델링 모드에서 직접 변형하는 방법**을 보여주는 영상.
  * 어떤 식으로 Dynamic Mesh Actor를 씬에 추가하고:

    * 모델링 모드 툴(Extrude, Cut, Weld 등)을 적용하는지
    * 스태틱 메쉬와 Dynamic Mesh의 차이가 무엇인지
  * “애셋 관리 줄이면서 레벨 안에서 바로 모델링”을 할 때 Dynamic Mesh가 왜 유용한지 설명.
* **추천 이유**

  * 질문처럼 “레벨에서 다이나믹 메쉬 잡고 바로 수정”하는 플로우를 보여줌.
  * Dynamic Mesh 전용 개념(라이브 업데이트, 고폴리 작업 시 이점 등)을 이해하는 데 좋음.
* **찾는 법**

  * 유튜브에서: `UE5 Modeling Mode Dynamic Mesh Actors` 검색.

---

## 3. 폴리곤 단위 변형(PolyEd 중심) – 세밀한 에디팅 튜토리얼

### 3) **UE5.1 PolyEd Modeling Part 1 – Selection** (Ryan Schmidt) ([유튜브][3])

> 이건 “변형”을 직접 다루는 영상이라기보다, 변형하기 전에 **선택/편집의 기초**를 다져주는 시리즈의 첫 파트라고 보면 됨.

* **내용**

  * 모델링 모드 안의 **PolyEd Tool** 사용법 시리즈 중 1편.
  * Dynamic Mesh/Static Mesh 둘 다 대상으로:

    * 버텍스/엣지/폴리곤 선택 모드
    * 마퀴 선택, 트랜스폼 기즈모 사용법
  * 이후 파트에서 Extrude, Inset, Bevel 같은 변형 툴도 다루기 때문에, 1편부터 쭉 보면 **“에디터 안에서 메쉬를 점·선·면 단위로 변형하는 전체 플로우”**를 볼 수 있음.
* **추천 이유**

  * PolyEd 툴은 UE5 모델링 모드에서 **폴리곤 모델링 느낌으로 메쉬 변형**하려면 필수 수준이라, 다이나믹 메쉬 작업에도 그대로 응용 가능.
* **찾는 법**

  * 유튜브에서: `UE5.1 PolyEd Modeling Part 1 Selection` 검색 후 같은 채널의 Part 2,3… 도 이어서 보기.

## 4. 문서지만, Dynamic Mesh 변형 툴 목록/개념 잡기에 좋은 공식 문서

영상은 아니지만, 다이나믹 메쉬를 모델링 모드에서 어떻게 다루는지 이해할 때 같이 보면 좋아서 넣어둘게.

### 4-1) **언리얼 엔진의 모델링 모드 (한국어 공식 문서)** ([Epic Games Developers][4])

* **내용**

  * 모델링 모드에서 지원하는 툴셋 개요:

    * Create / Transform / TriModel / PolyModel / Sculpt / Deform 등
  * 툴마다 **어떤 타입의 메쉬(Static, Dynamic, Volume)** 에 사용할 수 있는지 명시되어 있어서,

    * “이 변형툴이 다이나믹 메쉬에도 먹히는지” 확인할 때 좋음.
* **추천 이유**

  * 영상 튜토리얼에서 툴 이름이 휙휙 지나갈 때, 문서에서 **툴 이름으로 다시 찾아보면서 정리**하기에 딱 좋음.

## 5. 한국어 쪽 자료 & 블로그 기반 튜토리얼(영상 연계)

### 5-1) **[Unreal 02] Modeling Mode – 블로그 튜토리얼** (GAMEMING) ([GAMEMING][5])

* **내용**

  * UE5.0.3 기준 모델링 모드 기본 사용법을 한국어로 정리한 글.
  * 모델링 모드 진입, 플러그인 활성화, 기본 툴 설명과 함께 **공식 유튜브/강의 영상**도 같이 링크되어 있는 경우가 많음.
* **활용 방법**

  * 먼저 이 글로 모델링 모드 구조랑 메뉴 위치를 한국어로 이해 → 위에서 추천한 영어 영상 따라 할 때 훨씬 편해짐.
