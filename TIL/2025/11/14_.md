# 구현 목표 (한 BP에서 끝내기)

1. **열림 트리거 방식 선택**: 오버랩(자동문) 또는 상호작용(E 키 등)
2. **문 열림 방식 선택**: 슬라이딩(좌/우/상/하), 미닫이(슬라이딩의 변형으로 한쪽만 이동)
3. **문 크기 조절**: 스케일 기반 크기 프리셋/수치 입력
4. **문 열리는 시간 조절**: 타임라인 커브/시간으로 속도 제어 ([Epic Games Developers][1])

---

# BP 구성(필수 컴포넌트 & 변수)

**컴포넌트**

* `Root` (Scene)
* `SM_Frame` (문틀, 선택)
* `SM_Door` (문짝 StaticMesh)
* `Box_Trigger` (오버랩 트리거용)
* `Timeline_OpenClose` (열림/닫힘 애니메이션) ([Epic Games Developers][1])

**설정 변수(Expose on Spawn / Instance Editable)**

* `TriggerType` (Enum): `Overlap`, `Interact`
* `OpenMode` (Enum): `SlideLeft/Right/Up/Down`, `SlideSingle`(미닫이/한쪽만)
* `OpenTime` (float, s): 기본 1.0s (타임라인 길이와 동기) ([Epic Games Developers][1])
* `MoveDistance` (float, cm): 슬라이드 이동 거리
* `ScaleTarget` (Vector): 문 크기(스케일) 목표치
* `AutoCloseDelay` (float, s, 0이면 끔)
* `bStartsOpen` (bool)

# 논리 설계 (한 눈에 보는 플로우)

1. **입력/트리거 처리**

   * `Overlap` 선택 시: `Box_Trigger`의 **OnComponentBeginOverlap/EndOverlap** 이벤트로 열림/닫힘 호출. ([Epic Games Developers][2])
   * `Interact` 선택 시: 캐릭터의 라인트레이스 hit 결과가 문이면 `Interact` 입력 시 `ToggleDoor()` 호출(간단 라인트레이스 상호작용 튜토리얼 참조). ([Epic Developer Community Forums][3])

2. **애니메이션 구동**

   * `Timeline_OpenClose`에 `Alpha(0→1)` float 트랙 1개.
   * 매 틱 업데이트에서 `Lerp`를 이용해 위치/스케일 보간. ([Epic Games Developers][4])
   * **슬라이드/미닫이**: `SetRelativeLocation(SM_Door)`에 `StartLoc`→`TargetLoc` 보간 적용(축/방향은 `OpenMode`로 결정). 필요 시 Sweep 옵션으로 충돌 고려 가능. ([Epic Games Developers][5])
   * **크기 조절**: 액터 또는 문짝 컴포넌트에 `SetActorRelativeScale3D`/`SetRelativeScale3D`로 `ScaleTarget` 보간 적용. ([Epic Games Developers][6])

3. **시간/속도 제어**

   * `OpenTime`을 타임라인 길이로 사용(또는 `PlayRate = 1/OpenTime`). 커브를 바꾸면 가속/감속도 즉시 반영. ([Epic Games Developers][1])

4. **자동 닫힘**

   * 열린 후 `AutoCloseDelay > 0`이면 `SetTimer`로 일정 시간 뒤 `Reverse` 재생. (타임라인은 `Play`/`Reverse`로 양방향 제어) ([Epic Games Developers][1])

# 상세 구현 단계 (체크리스트 겸 가이드)

### A. 컴포넌트 배치

* `SM_Door`의 로컬 +X(또는 선택 축)를 “문 이동 축”으로 정의.
* `Box_Trigger`는 문의 앞쪽에 배치하고 **Generate Overlap Events** 켜기. (채널/프리셋은 캐릭터와 오버랩되도록) **OnComponentBeginOverlap** 바인딩. ([Epic Games Developers][2])

### B. 변수 초기화 & 프리컴퓨트

* BeginPlay에서 `StartLoc = SM_Door.RelativeLocation` 캐시.
* `OpenMode`에 따라 `TargetLoc = StartLoc + MoveAxis * MoveDistance` 계산. (Left/Right/Up/Down 분기)

### C. 타임라인 설정

* `Timeline_OpenClose`에 `Alpha(0~1)` float 커브 추가. 길이는 `OpenTime`.
* **Update**:

  * `Loc = Lerp(StartLoc, TargetLoc, Alpha)` → `SetRelativeLocation(SM_Door, Sweep=false)` ([Epic Games Developers][5])
  * `Sca = Lerp((1,1,1), ScaleTarget, Alpha)` → `SetActorRelativeScale3D(Sca)`(또는 도어만 스케일링하려면 `SM_Door.SetRelativeScale3D`) ([Epic Games Developers][6])
* **Finished**: 상태 플래그 업데이트(Opened/Closed)

### D. 트리거 타입 분기

* `TriggerType == Overlap`

  * BeginOverlap → `OpenDoor()`
  * EndOverlap → `CloseDoor()` (또는 딜레이 후 닫힘) ([Epic Games Developers][2])
* `TriggerType == Interact`

  * 캐릭터의 라인트레이스가 `BP_Door`를 Hit하면 `Interact` 입력에서 `ToggleDoor()` 호출. ([Epic Developer Community Forums][3])

### E. 퍼블릭 함수 (레벨에서 바로 쓰기)

* `OpenDoor()` → 타임라인 `Play`
* `CloseDoor()` → 타임라인 `Reverse`
* `ToggleDoor()` → 상태 보고 `Play/Reverse` 선택

### F. 디테일 패널 노출 항목

* `TriggerType`, `OpenMode`, `OpenTime`, `MoveDistance`, `ScaleTarget`, `AutoCloseDelay`, `bStartsOpen`

# 테스트 시나리오 (빠른 검증)

1. **Overlap 모드**: 캐릭터가 트리거에 들어가면 슬라이드/미닫이로 열리고, 나가면 닫힘. (커브가 매끄럽게 보간되는지 확인) ([Epic Games Developers][1])
2. **Interact 모드**: 문을 바라보고 E 입력 시 열림/닫힘 토글. 라인트레이스 거리/채널 점검. ([Epic Developer Community Forums][3])
3. **크기 조절**: `ScaleTarget`을 (1.2,1,1) 등으로 바꾸고 열림 애니메이션과 함께 변화하는지 확인. ([Epic Games Developers][6])
4. **시간 조절**: `OpenTime` 0.5s/2.0s로 변경하여 속도 차이 확인. ([Epic Games Developers][1])

# 미니 코드(노드) 가이드 (요점만)

* **Timeline_Update**

  * `Alpha` → `Lerp(StartLoc, TargetLoc, Alpha)` → `SM_Door.SetRelativeLocation`
  * `Alpha` → `Lerp((1,1,1), ScaleTarget, Alpha)` → `SetActorRelativeScale3D` ([Epic Games Developers][5])
* **Open/Close**

  * `OpenDoor()` → `Timeline.Play(PlayRate=1/OpenTime)`
  * `CloseDoor()` → `Timeline.Reverse(PlayRate=1/OpenTime)` ([Epic Games Developers][1])
* **Overlap 분기**

  * `BeginOverlap` → `if TriggerType==Overlap → OpenDoor()`
  * `EndOverlap` → `if TriggerType==Overlap → CloseDoor()` ([Epic Games Developers][2])
* **Interact 분기**

  * 캐릭터 BP에서 라인트레이스 → Hit가 `BP_Door`면 `ToggleDoor()` 호출 ([Epic Developer Community Forums][3])
