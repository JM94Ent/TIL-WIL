## 1. 공간 타입 목록 (예시 14종)

| HUB     | 허브        | 중앙 광장, 거점, 메인 동선 분기 지점        |

| COM     | 일반 전투     | 기본 전투가 일어나는 전투 구역             |

| SHOP    | 상점/거점     | 상점, 강화, 제작, 빌드 조정하는 구역        |
| PUZ     | 퍼즐        | 유대 퍼즐, 기믹 조작, 논리/공간 퍼즐 등      |
| CIN     | 연출/컷신 공간  | 이벤트, 컷신, 장면 전환, 스케일 보여주는 구역   |
| CHASE   | 추격/도망 구역  | 쫓거나 쫓기는 런 구간, 타이머/압박이 중심인 구역  |

| STEALTH | 잠입/회피 구역  | 들키면 위험한 은신 플레이 중심             |
| PLAT    | 플랫폼/점프    | 점프, 낙하, 이동기믹, 타이밍 액션이 핵심인 구역  |
| FARM    | 자원/파밍 구역  | 자원 채집, 몬스터 파밍, 반복 플레이를 고려한 구역 |
| SECRET  | 비밀/보너스 구역 | 숨겨진 방, 지름길, 숨겨진 보상 얻는 구역      |

# 4. 생각
공간을 설정하고 플레이 경험을 확실하게 줄 것이다라는 생각을 하지 않아서 레벨 자체가 흐지부지되는 경우가 생긴다.\
그에 대한 해결책으로 구간과 공간을 나눠 어떤 경험을 줄 것인가를 먼저 러프하게라도 계획하고 시작하기 위해\
타입과 태그를 나눴다.
