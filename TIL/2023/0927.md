# 특별강연 이력서/자소서 작성

강사= 김례진

1. 내 맘대로 정답
-  김동완 분석(혈액형, 고향, 형제, 주량, 생일)
-  성격별 유형분석에서 자신의 유형 중 쓸만한 것을 자소서에 적어라

2. 동물 버리기
-  본성(사자-자존심, 암소-일, 말-가족, 원숭이-친구, 양-사랑)
```
사자>원숭이>암소>말>양 선택
```
3. 채용분석
-  입사지원서 분석 (요즘엔 정형화된 지원서 양식이 없다)
-  자소서 (활동에 대한 경험)
-  NCS기반 능력 중심 채용 분석
```
-이력서
 개인을 객관적으로 이해하는 문서
 빈칸 자제

-자기소개서
 개인을 보다 깊이 이해하는 문서
 두괄식
 주장과 설득
```
 ( [NCS 사이트](https://www.ncs.go.kr/index.do) )
