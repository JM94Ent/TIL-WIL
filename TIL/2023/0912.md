## 1. 해야할 것
```
- 다리 구조물 및 흔들다리 제작
- 폐광 및 관리소 제작
```

## 2. 오늘 배운 것
```
- 주변 환경에 쓰일 에셋은 많은데 적절한 걸 찾기가 힘들다.
```


## 3. 개선
```
- 조금 이상하게 보이는 돌이랑 지형 랜드스케이프로 조절
- 폐광 및 관리소 제작하고 사이즈 조절
```

## 4. 느낀점
```
다만들고 나서 보니까 폐광이 맘에 안든다.
폐광 관리소도 좀 더 개연성 있게 에셋을 배치해야 할 것 같다.
주변 에셋 배치하면서 잘 꾸미면 좀 더 나아질 것 같다.
```
