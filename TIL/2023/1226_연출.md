# 4. 느낀점
```

우리 게임에 스토리의 비중도 꽤나 크게 보고있는만큼 생각은 계속하고 있지만 진행하기가 힘들다.
스토리의 초안을 내가 아닌 팀원이 짜준 이야기에 추가 설정을 만들고 있기 때문에 스토리 전반에 대한 분위기를 만들기가 어렵다.
```
