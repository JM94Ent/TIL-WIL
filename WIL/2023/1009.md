# 한 주 돌아보기
- 231002 ~ 231008
```
긴 명절 때문에 정신없는 시간이 가고 게임 취업 토크 콘서트에 참석도 하는 등 순식간에 지나갔다
교육은 언리얼엔진5와 다빈치 리졸브로 고퀄리티 영상을 제작하는 간단한 방법을 배웠다.

시퀀스를 이용한 영상촬영 기법은 동영상 편집과 비슷하지만 만져줘야할 부분들이 너무 많아서 시간이 걸렸다
영상은 이미 찍혀있는 영상을 편집하는 것 뿐이지만 시퀀스는 직접 그 현상을 구현해야하기 때문이다.
그래도 대충은 내가 원하는 영상을 만들어낼 수 있었다.
수업자체는 재미가 없었지만... 차라리 개인 목표를 세우고 구현한다는 주제를 줬으면 달라졌을지도?

개인 포트폴리오도 같이 정리하고 있는데 내 자신이 너무 부족하다는 걸 계속해서 느낀다.
좀 더 노력해야할 것 같다.
```
